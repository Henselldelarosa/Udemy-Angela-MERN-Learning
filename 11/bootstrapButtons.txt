Buttons
* Can be used by creating a *<button><button/>* html element
  and and giving it the class of *btn* and by adding the class
  for the color scheme you may want to choose
  1.<button type="button" class="btn btn-primary">Primary</button> => Blue
  2.<button type="button" class="btn btn-secondary">Secondary</button> => Gray
  3.<button type="button" class="btn btn-success">Success</button> => Green
  4.<button type="button" class="btn btn-danger">Danger</button> => Red
  5.<button type="button" class="btn btn-warning">Warning</button> => Yellow
  6.<button type="button" class="btn btn-info">Info</button> => Light Blue
  7.<button type="button" class="btn btn-light">Light</button> => White
  8.<button type="button" class="btn btn-dark">Dark</button> => Black
