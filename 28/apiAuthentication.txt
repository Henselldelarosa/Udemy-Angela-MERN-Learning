<API Authentication>
* Authenticating yourself with API provider

1. No Authentication
  * Usually have a limit to how many request could be made
  * Public API

2. Basic Authentication
  * You have an account with the API provider
  * Requires and email/username and password
  * Uses a base64 Encoding
3. API Key Authorization
  * Different from Authentication
  * When you Authenticate yourself is when you show you an user
  * Authorization is when you are authorize to view or do certain things
